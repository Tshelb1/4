Метод шифрования шифром табличной маршрутной перестановки

Аргументы
образецкласса "PermutationCipher", в котором устанавливается ключ
wstringэто строка зашифрования

Для начала вычисляется размер таблицы.

if (s.size()%w.k!=0) {
h=s.size()/w.k+1;
} else {
h=s.size()/w.k;
} // Вычисление размера таблицы.

Затем создаётся двумерный массив "wchar_t" Количество строк вычисляется по формуле, а количество столбцов это ключ, который устанавливается в классе "PermutationCipher".

wchar_t a[h][w.k];

После в созданный массив записываются символы строки зашифрования. Запись происходит по строкам.

for (int i=0; i<h; i++) {
for (int j=0; j<w.k; j++) {
if (k<s.size()) {
a[i][j]=s[k];
k++;
} else a[i][j]=' ';
}
}

После этого идёт сам процесс шифрования.

for (int i=0; i<w.k; i++) {
for (int j=0; j<h; j++) {
code+=a[j][i];
}
}
Возвращает
"code" это зашифрованная строка типа "wstring"
